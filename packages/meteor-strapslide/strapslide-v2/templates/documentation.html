HOW DOES IT WORK?

Strapslide is item based. Therefore, it uses list items for navigation. It has minimal Javascript use, therefore grants complete control to anyone with a little HTML and CSS knowledge.
All you have to do is to copy the markup or use one of the already made templates.

HTML STRUCTURE

Strapslide uses bootstrap elements as a markup. Our main container can be either simple, boxed or fluid . The fluid container allows it to take any size, being completely responsive. The boxed container will adjust according to the width of the span it's in.

We have a spanx structure which defines our slider width, spanning over x out of 12 columns. Each list element represents a slide. This is what interests you. Each slide has a background image and an content area.

You can add anything in the content area: text, images, videos, audio files, iframes, anything. The markup inside the content area div can be simple HTML or Bootstrap structured and is also real time responsive through media queries.

The slide which starts needs to have the class. 

<!-- STRAPSLIDE -->
    <div class="strapslide" id="strapslide">

        <!-- PRELOADER -->
        <div class="preloader text-center">
            <img src="images/loader.gif" alt="Loading.." id="loader" />
        </div>
        <!-- /PRELOADER -->

        <!-- STRAPSLIDE CONTAINER -->
        <div class="strapslide-container">

            <!-- SLIDE -->
            <div class="slide">
                <!-- SLIDE CONTENT -->
                <div class="slide-content container-fluid">

                </div>
                <!-- /SLIDE CONTENT -->
                <!-- BACKGROUND IMAGE -->
                <img src="images/background.jpg" class="background-image" alt=""/>
                <!-- /BACKGROUND IMAGE -->
            </div>
            <!-- /SLIDE -->

        </div>
        <!-- /STRAPSLIDE CONTAINER --> 

        <!-- PROGRESS BAR -->
        <div class="row-fluid">
            <div class="span12">
                <div class="progress progress-striped">
                    <div class="bar" style="width: 20%;"></div>
                </div>
            </div>
        </div>
        <!-- /PROGRESS BAR -->

        <!-- NEXT-PREV NAVIGATION -->
        <div class="row-fluid strapslide-nav">
            <a class="prev btn btn-large btn-primary" href="#">&lt;</a>
            <a class="next btn btn-large btn-primary" href="#">&gt;</a>
        </div>
        <!-- /NEXT-PREV NAVIGATION -->

        <!-- PAGINATION -->
        <div class="row-fluid strapslide-pagination">
            <div class="span4 offset4 text-center pagination">
                <ul>

                </ul>
            </div>
        </div>
        <!-- /PAGINATION -->

    </div>
    <!-- /STRAPSLIDE -->

CSS Files

Next, we link up our CSS file inside our head tag. The file can be found inside the strapslide / css folder. 

  <link href="strapslide/css/strapslide.css" rel="stylesheet" media="screen">


The CSS provided by Bootstrap is very recommended in order to use Strapslide at its full potential.

  <link href="strapslide/css-bootstrap/bootstrap-responsive.min.css" rel="stylesheet" media="screen">
  <link href="strapslide/css-bootstrap/bootstrap.min.css" rel="stylesheet" media="screen">

Important! Each of the background images used in strapslide must have the same size.
Warning! Strapslide .slider-container height must be set manually to prevent any issues. 
#strapslide .slider-container{ 
        height:820px; 
    }

JAVASCRIPT

First of all, we need to include the required javascript into our HTML page. All the required files can be found in the strapslide / js folder. We use the jQuery library together with the transit library for animations. The touchswipe and mousewheel plugins bring extra functionality. Strapslide.js puts them all together in a fully working plugin. 

<script src="http://code.jquery.com/jquery.js"></script><!-- jQuery Library -->
<script src="strapslide/js/jquery.touchSwipe.min.js"></script><!-- Mobile Swipe Feature -->
<script src="strapslide/js/jquery.transit.min.js"></script><!-- REQUIRED! CSS3 transitions -->
<script src="strapslide/js/jquery.mousewheel.js"></script><!-- Mousewheel Feature -->
<script src="strapslide/js/jquery.fitVids.js"></script><!-- Responsive Videos -->

<script src="strapslide/js/strapslide.js"></script><!-- Slider Plugin -->


We recommend including the Bootstrap library as well to use strapslide at it's full potential.

<script src="strapslide/js-bootstrap/bootstrap.min.js"></script>
<script src="strapslide/js-bootstrap/respond.min.js"></script>

Now that we have included the required files and we've set up the mark up, it's time to see what strapslide can really do. 

    $('#strapslide').strapslide({
        animation: "string",    //List of animations, example: "fade, rotatecw, scaleup, top"
        speed: 1000,    //Slide animation speed
        timeout: 5000,  //Slide display time
        responsive: 'keyword',   //Responsive mode, example: 'resize', 'after', 'none'
        autoplay: boolean,       //Autoplay feature, boolean means: true / false
        preload: boolean,        //Preload feature
        pauseOnHover: boolean,   //Pause On Hover feature
        pagination: boolean,     //Pagination feature
        mousewheel: boolean,     //Mousewheel feature
        keyboard: boolean,       //Keyboard Navigation feature
        touchscreen: boolean     //Touchscreen Enabled
    }, function(){
        //callback when slide enters
    }, function(){
        //callback when slide exits
    })


ADVANCED SETTINGS


Here are all the settings and features you can exploit with strapslide:

ANIMATION

Naturally, the question "What animations does it have?" appears. Luckily for you, strapslide comes with a suite of cool CSS3 animations. Any combination of these is possible in the animation input string/p>

  • 'fade' - Fade Animation
  • 'right' - Right Direction
  • 'left' - Left Direction
  • 'top' - Top Direction
  • 'bottom' - Bottom Direction
  • 'scaleup' - Scale up
  • 'scaledown' - Scale down
  • 'rotatecw' - Rotate Clockwise
  • 'rotateccw' - Rotate Counterclockwise

CALLBACKS

Each Caption can be animated directly with a function call and has it's own callback!

    callbackin:
    function(){
        if($('#strapslide .active').attr('id') == 'slideID'){
            $('#caption').fxenter("string",500);
    }

    callbackout:
    function(){
        if($('#strapslide .active').attr('id') == 'slideID'){
            $('#caption').fxexit("string",500);
    }

DATA-CONTROL

With strapslide, you're always unique. That's why we made it possible to set custom settings for each slide.

    <li class="slide active" id="slideID" data-animation="string" >
        <div class="container-fluid slide-content"></div>
        <img class="slide-background" src="path/to/image.ext"/>
    </li>
Example: 
data-animation="fade, left, scaleup, rotateccw"

CAPTION POSITIONING

Can be set in % or in px.

Caption positioning is a great feature. It allows you to position the elements based on the slide's current height.

<div data-left="number%" data-top="number%"></div>
<div data-right="number px" data-bottom="number px"></div>

Therefore, if data-top="10%" the div will remain at 10% from the top of the slide and responsively change position.
